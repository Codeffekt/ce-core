CeForms

Solution de formulaires pour l'acquisition et la gestion des données

CeForms est un ensemble d'outils logiciels OpenSource permettant de construire des solutions logicelles pour gérer l'acquisition et la restitution de données numériques.

Que contient CeForms ?

CeForms est un ensemble d'outils logiciels majoritairement écrit en Typescript qui comprend :

  • ce-events-data Les modèles (interfaces Typescript) pour la gestion des événements sur les formulaires
  • ce-core-data Les modèles (interfaces Typescript) relatifs aux formulaires
  • ce-core Une librairie Angular permettant de construire des interfaces Web (frontends) autour des formulaires

Qui utilise CeForms ?

CeForms est utilisé par les entreprise souhaitant simplifier les process nécessitant l'acquisition de données.
